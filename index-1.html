Moderation
made easy.

Every action DMs the user automatically with reason and details.

🔨 >ban

Permanently ban a user with a reason. Sends a DM before executing.

👢 >kick

Remove a user from the server. They can rejoin with an invite.

⏳ >tempban

Ban a user for a set time. Auto-unbans and notifies when it expires.

🔇 >to

Timeout up to 4 weeks. Sends a DM with duration and reason.

🗑️ >purge

Delete up to 100 messages at once. Auto-deletes confirmation.

⚠️ >warn

Log warnings per user. Track history with >warns command.
